# ميزة العداد التنازلي المتطور - نبض العقد

## النظرة العامة

تم تطوير واجهة متقدمة ومبتكرة لعرض العداد التنازلي للعقود مع عداد رقمي ديناميكي يعمل بشكل حي.

---

## الميزات الرئيسية

### 1. العداد الرئيسي (سنوات - شهور - أيام)

**التصميم:**
```
┌─────────────────────────────────┐
│         02 : 11 : 15            │
│        سنة  شهر  يوم            │
└─────────────────────────────────┘
```

**المميزات:**
- أرقام كبيرة وواضحة
- تنسيق tabular-nums للحفاظ على العرض الثابت
- فواصل `:` بين الوحدات
- تحديث تلقائي عند تغيير اليوم

---

### 2. العداد الرقمي الديناميكي (ساعات - دقائق - ثواني)

**التصميم:**
```
┌─────────────────────────────────┐
│    🕐  23 : 45 : 12  متبقي     │
└─────────────────────────────────┘
```

**المميزات:**
- عداد حي يتحرك كل ثانية
- تنسيق 00:00:00
- أيقونة ساعة
- عند وصول الثواني للصفر تنقص الدقائق ثم الساعات ثم الأيام

---

### 3. شريط المرحلة المحسّن

**بعد التحسين:**
```
┌─────────────────────────────────────────┐
│ [🌱]  مرحلة النمو    بداية رحلتك     │
│  (خلفية ملونة قوية - نص أبيض واضح)   │
└─────────────────────────────────────────┘
```

**المميزات:**
- خلفية بتدرج قوي (gradient)
- نص أبيض واضح مع ظل (drop-shadow)
- أيقونة داخل مربع شفاف
- سهل القراءة على جميع الخلفيات

---

## النتيجة النهائية
