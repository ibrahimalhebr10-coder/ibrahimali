## 6. الألوان والتصميم 🎨

```
🟠 Amber (ذهبي):
   - زر "السنوات المجانية"
   - الكود المميز
   - السنوات في الجدول
   - معاينة المكافأة

🟢 Green (أخضر):
   - رسائل النجاح
   - الحالة "مفعّل"

🔵 Blue (أزرق):
   - الحد الأدنى
   - معلومات إضافية

🟣 Purple (بنفسجي):
   - وصف الميزة

⚪ Gray (رمادي):
   - النصوص الثانوية
   - الحدود
```

## 8. التفاعل والحركات ✨

```
عند التحميل:
├─ الزر يظهر تدريجياً (fade-in)
└─ النافذة تنزلق من الأعلى (slide-down)

عند التحرير:
├─ الحقول تتمدد عند التركيز (focus)
└─ الأرقام تتغير بسلاسة (smooth)

عند الحفظ:
├─ زر الحفظ يدور (spinning)
├─ رسالة نجاح تظهر (fade-in)
└─ النافذة تُغلق تدريجياً (fade-out)
```
